# u-tools

A lightweight multi-tool website built with Nuxt 4 and Nuxt UI 4.

## Included tool

- Random name grouping: paste names line by line, set the number of groups, and generate balanced random groups.

## Tech stack

- Nuxt 4
- Nuxt UI 4
- Tailwind CSS
- @nuxtjs/i18n

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
